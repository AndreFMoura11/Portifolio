  • HTML5

    HTML (HyperText Markup Language) é a linguagem que estrutura páginas web usando tags para organizar títulos, parágrafos, imagens e links, definindo como o conteúdo é exibido nos navegadores.

  • CSS

    CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas web. Ela permite definir cores, fontes, espaçamentos e layouts, separando o conteúdo (HTML) da apresentação visual.

  • Python

    Python é uma linguagem de programação versátil e fácil de aprender, conhecida por sua sintaxe clara e legibilidade. É amplamente usada em desenvolvimento web, ciência de dados, automação, inteligência artificial e muitas outras áreas.
